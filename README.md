# Google Doogle Challenge

A collection of interactive, minimalist Google Doodle-like games built with Vanilla JavaScript, HTML5 Canvas, and pure CSS.

## 🎮 Available Doodles

- **Doodle Cricket**: A simple cricket game.
- **Doodle Ghost**: A spooky ghost-themed interaction.
- **Doodle Hurdles**: A fast-paced hurdles race.

## 🚀 How to Play

Each doodle is self-contained in its own directory:

1. Navigate to the doodle's folder (e.g., `doodle-cricket/`).
2. Open the `index.html` file in any modern web browser.
3. Use the root `index.html` as a central portal to access all games.

## 🛠️ Tech Stack & Rules

This project follows strict development principles:
- **Zero Dependencies**: Strictly Vanilla JS, HTML5, and CSS.
- **Doodle Aesthetic**: Clean, flat, and minimalist design using Google's primary colors (#4285F4, #EA4335, #FBBC05, #34A853).
- **Mobile-First**: Touch and mouse interactions are mapped to the same logic.
- **Single File Preference**: Doodles are kept as portable single-file implementations whenever possible.
